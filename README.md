[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

If you have docker and docker-compose
```bash
$ docker-compose up -d
```

Otherwise

Go to `app.module.ts` and replace mongodb url to 'mongodb://localhost/<dbname>'

```bash
$ npm install
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
